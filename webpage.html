Compiler For Dimished Java

Developed a compiler for the Diminished Java (DJ) programming language, using Flex to generate a lexical analyzer and GNU Bison to generate a parser. Wrote the regular expressions and tokens that described DJ and wrote the context-free grammar using those expressions/tokens. Implemented an AST-building parser (with GNU Bison), a symbol table used to develop the type-checker for DJ and a code-generator that converts DJ ASTs into Diminished Instruction Set Machine (DISM) programs; DISM is a RISC-style instruction set architecture.

https://github.com/MutaharS/DJ2DISM-Compiler
