Library Management System

Designed and developed a terminal-based library management system using Python and PostgreSQL. Patrons have the option to search for books by subject or an author's lastname. Librarians have authorization to check out books to users and process returns, as well as multiple reports available (book catalog, user report, overdue books report).

Database accesses performed via raw SQL queries. Passwords hashed prior to being stored into the database to prevent security hazards. Further security measures include: input sanitization to prevent SQL injection, the creation of PostgreSQL users with the corresponding priviliges of patron or librarion, and database views to limit data accessible to non-admin users.

Compiler For Diminished Java

Developed a compiler for the Diminished Java (DJ) programming language, using Flex to generate a lexical analyzer and GNU Bison to generate a parser. Wrote the regular expressions and tokens that described DJ and wrote the context-free grammar using those expressions/tokens. Implemented an AST-building parser (with GNU Bison), a symbol table used to develop the type-checker for DJ and a code-generator that converts DJ ASTs into Diminished Instruction Set Machine (DISM) programs; DISM is a RISC-style instruction set architecture.

A* Search Algorithm

Implementation of the A* search algorithm, an intelligent graph traversal algorithm for finding the shortest path from an initial node to a goal node. The algorithm uses a heuristic to guide its decisions on which paths to explore; the heuristic used here is the straight-line distance from a node to the goal node.
